# ТрендМет - Производство металлоконструкций

Современный коммерческий корпоративный сайт для ведущей компании по производству металлоконструкций полного цикла. Сайт с плавной анимацией, видео-фоном и адаптивным дизайном.

![TrendMet]

## Live Demo

[Посмотреть Демо](https://trend-met-store.vercel.app/) 

## Основные функции:

- Главная страница с видео-фоном и анимированной статистикой
- Каталог услуг - металлоконструкции, краны, мебель
- Портфолио проектов - реализованные объекты
- Склад продукции - нержавеющая, декоративная, окрашенная сталь
- Клиенты - логотипы партнеров
- Страница "О компании" с сертификатами
- Модальное окно для заявок
- Плавная прокрутка Locomotive Scroll

## Ключевые технологии:

- **Frontend:** React 18 + Vite
- **Routing:** React Router DOM v7
- **Анимации:** Framer Motion
- **Прокрутка:** Locomotive Scroll
- **Стили:** Tailwind CSS
- **Сборка:** Vite 6

## Установка и Запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/NikolayViktorovich/trend-met-store.git
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите dev-сервер:
```bash
npm run dev
```

4. Сборка для продакшена:
```bash
npm run build
```

5. Предпросмотр продакшен-сборки:
```bash
npm run preview
```

## Особенности реализации

- Использование Locomotive Scroll для плавной прокрутки
- Lazy loading видео-контента для оптимизации
- Анимированные счетчики с использованием requestAnimationFrame
- Адаптивная навигация с мобильным меню
- Модальные окна для форм обратной связи
- SEO-оптимизация с мета-тегами
